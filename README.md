# Red1dark Platform

Независимая технологическая экосистема нового поколения.

## 🚀 Возможности платформы

- **Мессенджер** — личные чаты, группы, супергруппы до 200 000 участников, публичные и приватные каналы
- **Боты платформы** — BotFather для создания ботов, Report Bot для модерации, RedVerify Bot для верификации
- **RedVerify** — официальная система верификации пользователей
- **CMS** — управление контентом без кода
- **Модерация AI** — автоматическая модерация через OpenAI Moderation
- **Панель управления** — токены ботов, API-ключи, история действий, привязанные аккаунты
