# Red1dark Platform

Red1dark Platform — независимая технологическая экосистема, включающая сайт, мессенджер, систему ботов, верификацию RedVerify, роли пользователей и панель управления.

## 🚀 Основные возможности

### 🔹 Два профиля
1. **Профиль сайта (админский)**  
   — роли, доступы, токены, API-ключи, настройки, RedVerify статус.

2. **Профиль мессенджера (публичный)**  
   — ник, аватар, о себе, каналы, группы, статус RedVerify.

### 🔹 Мессенджер
- личные чаты  
- группы до 200  
- супергруппы до 200000  
- каналы (публичные/приватные)  
- боты  
- модерация через OpenAI Moderation  

### 🔹 Официальные боты
- **BotFather** — создание и управление ботами  
- **Report Bot** — жалобы  
- **RedVerify Bot** — верификация пользователей  

### 🔹 Боты платформы
- создание токена  
- управление через панель  
- код пишется в VSCode  
- запуск локально или на внешнем хостинге  
- поддержка Telegram интеграции  

### 🔹 RedVerify
- официальная верификация  
- значок отображается только у проверенных  
- значок: https://imgfy.ru/ib/vpttRAnmBDZi4up_1776534392.webp
